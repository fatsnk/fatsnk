v1.27 新功能亮点

API 模板引擎、MCP 工具调用、配置档系统、数据备份等新功能上线，全面提升 AI 交互体验。

核心

API 模板引擎

通过 JSON 配置文件描述任意 LLM API 的请求与响应结构，自动处理 URL 拼接、请求头构建、采样参数映射与流式解析。开箱支持 OpenAI 兼容格式，也可导入自定义模板适配任意 API。

核心

MCP 服务与工具系统

支持连接 Streamable HTTP 协议的 MCP 服务，自动获取工具列表并注册。内置内容追加和改写工具；在提示词中添加工具说明后，任何大模型都可在聊天中调用已注册的工具，不依赖模型自身的函数调用能力。

新增

配置档系统

将预设、全局世界书、全局正则等配置的开启状态保存为配置档，在聊天中一键切换不同的环境配置，快速适应不同的对话场景。

新增

数据备份与迁移

支持 API 配置的导入导出，以及应用内所有数据和重要配置的完整备份。配合配置档功能，轻松实现数据迁移和环境还原。

增强

聊天图片查看器

点击聊天气泡下方按钮放大查看原图，文生图时可长按触发大图预览，新增图片保存功能。支持在聊天菜单直接发送本地图片。

增强

快速切换 API 与模型

在聊天界面即可快速切换 API 配置和模型，支持备选模型输入框快速扩充模型列表，无需离开对话即可调整。

增强

聊天记录拆分

长按聊天记录即可触发拆分功能，适用于处理超大型聊天存档，让记录管理更加灵活高效。

优化

流式传输优化

提升了流式响应的稳定性和兼容性，优化了流式思考内容的解析与展示，带来更流畅的对话体验。

核心优势

本地存储保障数据安全，兼容 SillyTavern 文件格式，高度可定制的个性化体验。

本地数据存储

所有数据保存在本地，内置存储管理与数据备份，可导出应用中的全部资料，数据安全有保障

SillyTavern 兼容

支持导入导出 SillyTavern V2 格式的角色卡、预设、世界书、正则表达式与聊天记录

个性化定制

导入本地字体、更换聊天背景、调整气泡颜色与字体样式，保存主题配置随时切换

角色管理

直观的角色编辑界面，便捷的收藏与管理功能，长按头像即可快速进入编辑
